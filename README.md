Exercícios práticos de visualização e análise de dados em Python, desenvolvidos durante o curso Fundamentos de Linguagem Python Para Análise de Dados e Data Science da Data Science Academy (DSA).

📈 Matplotlib — Visualização de Dados
DSA-Python-Cap11-Matplotlib.ipynb
Exercícios práticos com a principal biblioteca de visualização do Python:

Criação de gráficos de linha, dispersão, barras e histogramas
Plotagem de funções matemáticas (ex: parábolas com raízes e vértice)
Configuração de títulos, labels, cores e estilos
Exportação de figuras em diferentes formatos

🎨 Seaborn — Visualização Estatística
DSA-Python-Cap11-Seaborn.ipynb
Exercícios com a biblioteca de visualização estatística construída sobre o Matplotlib:

Análise exploratória de datasets reais
Gráficos estatísticos: boxplot, heatmap, pairplot, violinplot
Integração com Pandas DataFrames
Personalização de paletas e estilos
